# Assets Folder

This folder contains static assets for the Mykonos luxury e-commerce website.

## Structure

- `images/` - Image assets including favicon, logos, and other graphics
  - Place your `favicon.ico` here
  - Place your logo files here (SVG, PNG formats)
  - Product images should be uploaded to Supabase storage

## Favicon

To add a favicon:
1. Place your `favicon.ico` file in `/public/assets/images/`
2. Update the reference in `app/layout.tsx` if needed

Recommended favicon sizes:
- favicon.ico (16x16, 32x32, 48x48)
- apple-touch-icon.png (180x180)
- favicon-32x32.png
- favicon-16x16.png
